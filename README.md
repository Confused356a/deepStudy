# DeepStudy — 深度学习论文复现与思考

研究生阶段系统学习深度学习：经典论文复现、架构思考、实验记录。基于 PyTorch 生态。

## 目录结构

```
deepStudy/
├── papers/          # 按论文组织的复现代码
│   └── <paper-slug>/
│       ├── README.md    # 论文要点、复现笔记
│       ├── model.py     # 模型定义
│       ├── train.py     # 训练脚本
│       └── notes/       # 推导草稿、实验日志
├── playground/      # 临时实验、玩具代码
├── utils/           # 跨论文复用的工具
└── datasets/        # 数据集（不提交）
```

## 学习原则

- 先理解动机和核心思想，再抠细节
- 能跑通 > 完美复现；先跑起来，再对齐精度
- 每个论文复现后记录"我学到了什么"
